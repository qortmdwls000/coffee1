📦 Step. 1 : HTML 필수 태그와 속성(특징)

  1. html의 구조 (html, head, body)
  2. 들여쓰는게 무슨 의미일까? (부모-자식)
  3. 태그의 사용법 (태그는 반드시 닫는 태그(/)가 있어야 한다? No)
  4. <head>에 들어가는 태그들 (meta, title, link ....)
  5. 반드시 알아두어야 할 태그들
    1. <h>(반비례)
    2. <div>, <span> (차이점)
    3. <p>, <br> (서로 차이점도 설명 및 p태그와 div태그와의 차이점 설명)
    4. <a> (반드시 href 속성이 있어야 한다. 속성의 default는 "#")
    5. <img /> (반드시 src 속성이 있어야 한다), (닫는 태그가 필요없다)
    6. <input /> (type 속성별 차이점), (닫는 태그가 필요없다)
    7. <button>
    8. <table>, <tr>, <td>
    9. <ol>, <ul>, <li>

📂 Step. 2: 파일의 경로설정

  1. "/" : 루트
  2. "./" : 현재폴더
  3. "../" : 현재폴더의 상단폴더
    • HTML과 CSS을 연결할 때, 파일의 위치가 중요
    • 경로만 제대로 잡아도 오류의 반은 줄일수 있답니다.

📱 Step. 3 : CSS 머지?

  1. ✅ CSS 적용 안될 경우?

    2. ✅ CSS 적용 안된 경우 ✅ CSS 적용 된 경우
    ✔ HTML의 <head> 태그에서 CSS 파일이 연결되어 있는지 확인!
    예: <link rel="stylesheet" href="style.css">

  2. ✅ CSS 기본 문법

    3.             div {
              color: blue;
            }
          
                table {border-collapse: collapse;}

            table,th,td {
                border: 2px solid #333; /* 테두리 + 실선 */
            }
    • { } 중괄호 안에 스타일 작성
    • ;는 각 속성 선언의 끝에 붙임
    • 선택자 키와 값

  3. ✅ CSS 적용 방법

    • HTML 태그에 직접 적용: <div style="color:blue">
    • ID 선택자 (#): #myId { ... }
    • Class 선택자 (.): .myClass { ... }

  4. ✅ ID vs CLASS 차이

    • ID: 문서 내에서 한 번만 사용
    • CLASS: 여러 요소에 반복 사용 가능

  5. ✅ 선택자 종류

    • 복합선택자: 여러 조건 조합
    • 일치선택자: div.style { }
    • 자식선택자: div > .style { }
    • 후손선택자: div .style { }
    • 스타일을 정교하게 지정하고 싶을때 선택자를 잘 선택해서 사용

  6. ✅ 자주 쓰이는 스타일 속성

    • color: 글자 색
    • background-color: 배경 색
    • border /border-radius: 테두리 / 둥글게
    • solid: 테두리 실선
    • width / height: 너비 / 높이
    • display: flex: 레이아웃 정렬
    • justify-content: 주축 정렬
    • align-items: 교차축 정렬
    • flex-direction: 방향 설정
    • font-family: 글꼴 지정
    • text-align: 글자 정렬
    • margin/padding: 바깥 여백 / 안쪽 여백
    • float: 요소를 띄움
    • text-decoration: 밑줄, 취소선 등

  7. ✅ hover
    • :hover: 마우스를 올렸을 때 효과

  8. ✅ position 속성
    • static: 기본값 (기준 없음)
    • relative: 자기 기준 위치 이동
    • absolute: 가장 가까운 부모 기준
    • fixed: 화면 기준 (스크롤에도 고정)
    • sticky: 스크롤하다가 특정 위치에서 고정

  9. ✅ 단위 종류
    • px: 픽셀
    • in: 인치
    • %: 상대 비율
    • vh: 화면 높이 기준 비율

  10. ✅ 미디어쿼리 (반응형 웹)

    11.             @media (max-width: 600px) {
              body {
                background-color: lightgray;
              }
            }

💡 Step. 4 : 웹사이트 분석 방법 (화면 설계)

  • 웹사이트 화면(프론트엔드) 코딩에는 정답이 없음. (여러가지 태그 조합으로 어떻게든 만들면 됨)
  • 클론코딩의 뜻과 클론코딩을 하기위한 (크롬)개발자도구 활용 방법(clone:복제하다)
  • 인터넷에 있는 이미지를 내 웹사이트에 적용하는 방법
  • 연습문제 : 자신이 원하는 웹사이트를 접속하여 개발자 도구를 이용하여 화면을 박스로 나누고 구조를 파악하기 (패드, ppt or 종이)
